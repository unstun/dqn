# DQfD references (papers + surveys)

This folder contains the **literature references** used to implement and write about the repository's
**strict DQfD (Deep Q-learning from Demonstrations)** baseline.

## Why these PDFs look a bit different

In this environment, shell tools (`wget`/`curl`) cannot access the internet, so we cannot directly download the
original arXiv PDF files. Instead, we use **ar5iv** (arXiv â†’ HTML) and print the HTML to PDF via Playwright.

As a result:
- These PDFs are **offline-readable** and preserve equations reasonably well.
- They may **not be identical** to the original arXiv PDF formatting/pagination.

The authoritative sources are the arXiv pages linked below.

## Repro notes (how these PDFs were generated)

The PDFs were generated by opening the ar5iv HTML page and printing it:

```js
await page.goto("https://ar5iv.labs.arxiv.org/html/<ARXIV_ID>", { waitUntil: "networkidle" });
await page.waitForTimeout(1000);
await page.pdf({
  path: ".../<ARXIV_ID>_title.pdf",
  printBackground: true,
  format: "A4",
  margin: { top: "12mm", bottom: "12mm", left: "12mm", right: "12mm" },
});
```
